# Group-Face-Recognition-System

##  Project Description
This project is an AI-based Attendance System using Face Recognition.

##  Technologies Used
- Python
- Flask
- OpenCV
- InsightFace
- NumPy
- Scikit-learn

##  Features
- Face Detection
- Face Recognition
- Automatic Attendance Marking
- CSV Record Storage

## How to Run
1. Install requirements
pip install -r requirements.txt

2. Run the app
python app.py
